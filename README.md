# TrendTracker

自动追踪 Reddit、Product Hunt 等平台的产品趋势，生成多维度洞察报告。

## 🌐 在线演示

- **产品官网**: [Landing Page](https://matrix-wong-ai.github.io/trend-tracker/landing.html)
- **开发历程**: [Skill 开发可视化](https://matrix-wong-ai.github.io/trend-tracker/)

## ✨ 核心功能

- 自动数据采集（Product Hunt + Reddit）
- 多维度趋势分析
- 可视化 HTML 报告
- 深色主题设计

## 📄 文件说明

- `index.html` - Skill 开发流程可视化页面（skillvisual 风格）
- `landing.html` - 商业化产品 Landing Page（日式简约设计）

## 🚀 技术栈

- Product Hunt API（Token 认证）
- Reddit RSS Feed
- Python 数据抓取
- HTML/CSS 可视化
