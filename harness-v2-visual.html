02
v2.0 核心架构：三层进化
ROUND 1: v1.0 问题
质量管控 ≠ 价值创造

v1.0 能拦住错误，拦不住平庸。合规的文章读完毫无收获。

ROUND 2: v2.0 方案
对抗式创作系统

Generator 和 Evaluator 之间有张力，推动内容从"合格"走向"有洞察"。

ROUND 3: 核心机制
LAYER 0
Light Input · 轻量输入
链接 + 角度 + 类型。输入减负，让 Planner 扛重活。
LAYER 1
Planner · 策划者
抓取、提取、评分、缺口分析、结构规划。自动生成大纲。
LAYER 2
Sprint Contract · Sprint 合同
写前对齐验收标准。把"写完发现方向不对"前移到"动笔前"。
LAYER 3
Generator · 执行者
按合同分段执行，只使用 Planner 提供的事实。
LAYER 4
Evaluator · 审核者
硬性规则 + 洞察度判断。查"有没有认知增量"。
LAYER 5
Human-in-the-Loop · 人在回路
事实确认、风格校准、发布决策。
